## 1. Yay setup
```sh
sudo pacman -S --needed git base-devel
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si
yay -Y --gendb
yay -Syu --devel
yay -Y --devel --save
```

## 2. Setting display on Hyprland
```sh
nano .config/hypr/hyprland.conf
```
```conf
# make command or remove
autogenerated = 1
# monitor settings
monitor=,1920x1080@165,auto,1.5 # check monitor with hyprctl monitors
```

## 3. Install Hyprland packages
```sh
sudo pacman -S swww hyprpaper zsh rofi waybar swaync grim --needed
```
